The UNSW Search Box module provide a block that allows choosing to search 
either the current site or all UNSW websites with the help from 
Google Custom Search according to the brand guideline.


Installation
------------

1) Copy the unswsearchbox folder to the modules folder in your installation.

2) Enable the module using Administer -> Modules
   (/admin/modules).

3) Enable the "UNSW Search Box" in the Block Administration page. Visit Administer ->
   Structure -> Blocks, then drag the block to the region that you want.
